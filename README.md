##### 1.

`Tab` vs `Space`

> `Tab`

> Space키를 여러 번 사용하는 것과 Tab을 한 번만 사용하는 것의 결과가 같다면 동작이 심플한 Tab을 주로 쓰곤 합니다.

`세미콜론 O` vs `세미콜론 X`

> `세미콜론 O`
> 코드의 마지막 부분이라는 것을 명확하게 알 수 있기 때문입니다.
